What 'scope' is, and how it works in JavaScript

Declaring a variable means that it is visible in a certain part of the code. If it is declared outside a function it is said to have global scope ie it is visible and can be used anywhere in the program. If it is declared within a function than it is said to have local scope so it is visible and can be used within the function only.
