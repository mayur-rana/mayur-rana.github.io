What is grid based design?

Websites can be broken down into grids. These grids can be broken down further into smaller and smaller grids. Arranging these grids in a way that takes advantage of how our eyes navigate a page (from left to right, top to bottom) is called the grid pattern.

Whats all the hype about responsive webpages?

With different screen sizes/resolutions, responsive webpages are a very convenient way for the layout to adapt automatically to the size of the screen or browser without having to have multiple layouts for different sizes.

What is semantic structure?

It is using semantic tags in a structured way that makes it easy for humans and the computer to read, follow along and make sense of it.
